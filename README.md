## Setup and Execution
The code in this repository is written in Python 3 and structured using Jupyter Notebooks. To run the files locally:
1. Ensure Python 3 and Jupyter are installed on your machine.
2. Clone this repository.
3. Launch Jupyter Notebook or Jupyter Lab and open the `.ipynb` files.
